========== ENDPOINTS REQUERIDOS ===========

-------- GENERALES ---------
1. iniciar sesión
2. cerrar sesión

-------- ADMINISTRADOR -----------

1. archivo auth.ayd para autenticación 
2. aceptar/modificar estado de empresas
3. ver empresas
4. ver usuarios
5. modificar estado de usuarios (dar de baja)
6. generar reportes



-------- USUARIO ------------
1. crear cuenta
2. visualizar datos de puestos laborales
3. buscar empleos por área
4. consultar a la tabla de PUESTO toda la información y mostrarla
5. cargar cv 
6. ver el listado de aplicaciones conforme el id_solicitud
7. actualizar el estado de postulación en la tabla SOLICITUD_EMPLEO


-------- EMPRESA ----------
1. visualizar ofertas laborales activas
2. atender aplicación ---> Cuando se visualiza el cv debe actualizarse el estado
3. publicar puesto de trabajo
4. modificar estado del puesto de trabajo ---> por si ya se contrató a alguien y la plaza ya no está disponible
5. visualizar el historial de puestos de trabajo
6. contactar vía correo electrónico a los aplicantes
